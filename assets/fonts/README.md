# Fonts

Download Outfit from Google Fonts: https://fonts.google.com/specimen/Outfit

Download these 4 weights and place them in this folder:
- Outfit-Light.ttf
- Outfit-Regular.ttf
- Outfit-Medium.ttf
- Outfit-SemiBold.ttf

Or run this script from the project root:
  npx expo install expo-font
  # Then manually download from fonts.google.com/specimen/Outfit
  # Select: Light (300), Regular (400), Medium (500), SemiBold (600)
  # Download family -> extract -> copy the 4 TTF files here
